# Changelog

## [1.0.0] - 2026-04-13

### Added
- Two-entity architecture: Sliders (parent) and Slides (children) with full admin CRUD
- Admin grids and forms for managing sliders and slides
- Responsive image support with separate desktop, tablet, and mobile uploads
- Rich HTML content overlays with WYSIWYG editor
- Configurable slider settings: autoplay, speed, transition effect, loop, arrows, dots, pause-on-hover
- Widget integration for placing sliders on any CMS page or block
- Hyva theme support with Alpine.js-based slider and Luma support with RequireJS
- Touch/swipe navigation for mobile devices
- Keyboard navigation (arrow keys) for accessibility
- Date-based slide scheduling (start/end dates)
- Store view scoping for sliders and slides
- Sample data with pre-built slider configurations and demo images
- CSS custom property theming for easy visual customization
- ACL resources for granular admin permissions
- Image upload via AJAX with drag-and-drop support in admin forms
